## This App

This app demonstrates a simple photo display using Streamlit and Pillow. It is a template for adding any image to a web application deployed on Streamlit Cloud.

### How to run locally

1. Clone the repository  
2. Install dependencies: `pip install -r requirements.txt`  
3. Run the app: `streamlit run app.py`

### Live Demo

After deployment on Streamlit Cloud, the app will be available at a public URL.
